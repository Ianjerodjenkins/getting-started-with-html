My First HTML5 Web Page Definitions:

HTML: (Hypertext Markup Language) is the set of markup symbols or codes placed in a file that is intended for display on a web page.

HTTP: (Hypertext Transfer Protocol) communicates between the client computer and the web server

World Wide Web: a graphical user interface providing access to information stored on web servers connected to the internet

Web Client: a). usually runs web browser b). uses HTTP c). requests web pages from a web server d). Receives web pages from a web server

Web Server: a). continually connected to the internet b). runs web software c). receives a request for the web page d). responds to the request and transmits the status code, web page, and associated files

Symbols:

DOCTYPE: !DOCTYPE HTML-this symbol usually begins an HTML document

Title: (title)(/title)-the title of the web page is found here.

Body: (body)(/body)-most of the website is composed between these two tags

Page Break: (p)(/p) or (br)(/br)-these two symbols will create a page break

Image: (img src="url" alt="text")-creates a hyperlink on a webpage

HTML Attributes: (tag attr="value")contents(/tag)
